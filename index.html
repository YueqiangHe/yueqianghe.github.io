Dense kernel-based LU solver for Kunpeng Processor

Advisors: Prof. Hong An & Prof. Junshi Chen

• Project Introduction: We are committed to researching LU solvers. Traditional solvers struggle with memory limits and inefficient handling of locally dense regions. We propose BTLU, a new LU factorization method that directly splits matrices into uniform dense blocks, turning memory-bound tasks into compute-intensive ones. This approach better matches modern CPU architectures and achieves significant speedups on Intel, AMD, and ARM platforms.
• Research Methods: Our method combines RCM reordering to cluster nonzeros near the diagonal with a block-tile structure that organizes data into cache-friendly dense tiles and parallelizable blocks. Factorization is performed using efficient level-3 BLAS kernels, while OpenMP ensures balanced multithreading. Unlike traditional approaches requiring symbolic analysis, BTLU dynamically allocates tiles, reducing memory use and improving NUMA locality. This design maximizes cache efficiency and scalability.
• My Contribution: 1. Optimization tests on various platforms 2. Try to solve the NUMA effect of existing parallel algorithms 3. Comparison and reflection of the existing work with previous work
• Relevant Paper: Uniform Dense Blocking for Efficient Sparse LU Factorization in First-Principles Materials Simulation. In: Euro-Par 2025: Parallel Processing. Lecture Notes in Computer Science, vol. 15902, pp. 341–354. Springer, Cham (2026). DOI: 10.1007/978-3-031-99872-0_24
